# Lab3 File Comparator

Консольное Java-приложение для рекурсивного сканирования директорий и сравнения их состояния (Snapshots).
Позволяет отслеживать изменения в файловой системе (какие файлы были добавлены, удалены или изменены), используя хеширование (MD5/SHA).

## Функционал
- **Режим Scan**: Сканирует указанную папку, считает хеши файлов и сохраняет результат в JSON-файл.
- **Режим Diff**: Сравнивает два JSON-снимка и выводит отчет об изменениях.
- **Конфигурация**: Настройка алгоритма хеширования и игнорируемых расширений через файл `app.properties`.

## Требования
- Java 21+
- Любая ОС (Windows, Linux, macOS)

## Установка и Запуск

### 1. Скачивание
1. Перейдите на вкладку **Actions** в репозитории GitHub.
2. Откройте последнюю успешную сборку (зеленая галочка).
3. Внизу в разделе **Artifacts** скачайте архив `app-jar-windows-latest` (или ubuntu).
4. Распакуйте файл `.jar` в любую папку.

### 2. Настройка
Рядом с `.jar` файлом создайте файл `app.properties` со следующим содержимым:

```properties
hash.algorithm=MD5
scan.ignore.patterns=*.log,*.tmp,*.bak
